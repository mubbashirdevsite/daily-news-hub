🚀 Daily News Hub

A modern and responsive News Web App built with HTML5, CSS3, JavaScript, jQuery, and NewsAPI. Browse the latest headlines across multiple categories, search for news by keyword or country, and enjoy a beautiful dark/light mode experience.

📁 Project Structure
daily-news-hub/
├── index.html      # Main UI structure
├── style.css       # Styling and responsive design
├── script.js       # App logic and API integration
└── README.md       # Documentation
✨ Features
📰 Latest news powered by NewsAPI
🔍 Live search with debounce
📂 Category filtering
Technology
Business
Sports
Health
Entertainment
🌍 Country-based news search
🌙 Dark / Light mode
📱 Fully responsive design
⚡ Loading skeleton animations
🎨 Modern glassmorphism UI
🔄 Demo fallback data when API is unavailable
🛠️ Tech Stack
Layer	Technology
Markup	HTML5
Styling	CSS3
Logic	Vanilla JavaScript (ES6+)
Library	jQuery
API	NewsAPI
Fonts	Google Fonts (Inter)
🚀 Getting Started
1. Clone Repository
git clone https://https://github.com/mubbashirdevsite/
2. Open Project

Simply open index.html in your browser.

Or use VS Code Live Server:

Install Live Server Extension
Right-click index.html
Select Open with Live Server
🔑 NewsAPI Setup
Create an account at:
https://newsapi.org
Get your API key.
Replace this line inside script.js:
const apiKey = 'YOUR_API_KEY';

with

const apiKey = 'YOUR_NEWSAPI_KEY';
